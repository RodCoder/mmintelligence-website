# Kinari Marketing Landing Page - Claude Context

## Figma MCP
- Figma MCP is installed and configured in `~/.claude.json` with token authentication (`X-Figma-Token` header)
- To push UI to Figma: build/preview the project in browser, then say "Send this to Figma"
- If Figma MCP shows disconnected, run `/mcp` to verify status and re-authenticate if needed

## Tools & Setup
- Node via nvm (v20.14.0)
- Claude Code installed globally via npm
